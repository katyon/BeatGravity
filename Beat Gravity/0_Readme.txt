///// かちょ製StageEditorの取扱説明書 /////

【使い方】
1.「0_StageEditor.exe」を起動する(0_に特に意味はなく、並び替えで上の方にしたかっただけ)
2.作るステージを選択する(1〜4まで)
3.作る前にステージを初期化する(全て0の描画しないチップにする)かどうか選択する
　「Yes」を選ぶと初期化され、「No」を選ぶとステージのデータを引き継いで作業できる
4.ステージを作成する(操作についは画面に表示している)
5.「Ctlr + z」でデータを保存
6.「Esc」で終了

【注意点】
・ステージを残しておきたい場合は「Data→Stages→「残しておきたいステージ」.txt」をファイルの外に出しておくと残せる
・外に出した場合、次にステージを作成する際には必ず初期化して始めること（初期化しなきゃ落ちちゃうぞ☆）
　例：「stage2.txt」を外に出して、次に「stage2」を作成する時は初期化の画面で「Yes」を選ぶ
・プレイヤーの縦(Y)の初期座標は真ん中にしている
・縦のゲーム画面の大きさはマップチップ18枚分、横は32枚分(1920×1080)
・縦はゲーム画面の3倍の枠(54枚分)をとっているので、作成する座標に気を付けること
・横は1700枚分。曲が終わるぐらい
・注意点とは？？？


※もし間違えて初期化で始めてしまった等の時はデータを保存(「Ctlr + z」)せずに「Esc」を押せばなかった事にできる
※なにか分からないことやバグがあればかちょまで
